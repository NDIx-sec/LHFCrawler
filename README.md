<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'># <span
class=SpellE>LHFCrawler</span> (Low Hanging Fruit Crawler)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>##
Overview<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span class=SpellE><span style='mso-fareast-font-family:
"Times New Roman"'>LHFCrawler</span></span><span style='mso-fareast-font-family:
"Times New Roman"'> is a Python-based pipeline for automated domain
reconnaissance and reflected XSS vulnerability testing. It queries public
certificate transparency logs (crt.sh), gathers URLs from discovered domains
using multiple open-source tools, and scans those URLs for reflected XSS
vulnerabilities using custom payloads.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>In
addition to automatic domain and URL discovery, <span class=SpellE>LHFCrawler</span>
also supports manually defined individual domains for targeted and automated
domain reconnaissance and reflected XSS vulnerability testing. <o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>##
Key Features<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Collects domains from crt.sh using wildcard patterns<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Detects live domains via HTTP/HTTPS status code checks<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Collects URLs via:<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>  </span>- `<span class=SpellE>gau</span>` (<span
class=SpellE>GetAllUrls</span>)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>  </span>- `<span class=SpellE>waybackurls</span>`
(Internet Archive)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>  </span>- `<span class=SpellE>hakrawler</span>`
(JavaScript-aware crawler)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Injects XSS payloads into URL parameters<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Detects reflections of payloads in responses<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Multithreaded for high performance<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
JSON and HTML report generation<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
CLI-friendly with detailed argument options<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>##
Installation<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>###
Prerequisites<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Python 3.7 or newer<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>- Go
(required for URL collection tools)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>###
Installation Steps<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>1.
Clone the repository:<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```bash<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>git

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```bash<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>3.
Install Go-based tools:<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```bash<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>go
install github.com/lc/<span class=SpellE>gau</span>/v2/<span class=SpellE>cmd</span>/<span

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>##
Usage<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>###
Test Specific URLs<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>You
can test custom URLs directly by providing a domain with a ``` --target ```
flag or pointing the tool to a local file or custom URL feed source.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>This
is useful for:<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Retesting URLs found in a previous scan<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Manually collected or known vulnerable endpoints<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>_Example
usage will depend on the specific integration or customization — plugin support
for this is part of future plans._<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

Basic Usage<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```bash<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>python3
LHFCrawler.py --<span class=SpellE>tld</span> hu --limit 20<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>python3
LHFCrawler.py --target example.com --payloads payloads/XSS/blind.txt<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>##
Options<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```bash<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--threads<span
style='mso-spacerun:yes'>         </span>Number of threads (default: 10)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--limit<span
style='mso-spacerun:yes'>           </span>Maximum number of domains to process
(default: 50)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--out<span
style='mso-spacerun:yes'>             </span>Output JSON report file name<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--<span
class=SpellE>tld</span><span style='mso-spacerun:yes'>            
</span>Domain suffix to scan (e.g. hu, com, <span class=SpellE>xyz</span>, all)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--tools-check<span
style='mso-spacerun:yes'>     </span>Checks if required tools (<span
class=SpellE>gau</span>, <span class=SpellE>hakrawler</span>, etc.) are
installed<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--no-check-cert<span
style='mso-spacerun:yes'>   </span>Skip SSL certificate verification<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--use-cache<span
style='mso-spacerun:yes'>       </span>Use cached crt.sh responses<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--target<span
style='mso-spacerun:yes'>          </span><span class=SpellE>Target</span> a
specific domain<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--payloads<span
style='mso-spacerun:yes'>        </span>File containing XSS payloads<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--max-errors<span
style='mso-spacerun:yes'>      </span>Max errors per domain before skipping
(default: 5)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--max-hits<span
style='mso-spacerun:yes'>        </span>Max XSS hits per domain (default:
unlimited)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>--html-report<span
style='mso-spacerun:yes'>     </span>Generate HTML report (auto or custom
filename)<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>```<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

Payload Format<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>##
Performance Tips<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Increase `--threads` for faster scanning on multi-core systems<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
Use `--limit` to avoid scanning too many domains<o:p></o:p></span></p>

Use caching (`--use-cache`) for faster repeat runs<o:p></o:p></span></p>

Performance Options<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
`--timeout 5`: request timeout in seconds<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>-
`--delay 0.1`: delay in seconds between each request<o:p></o:p></span></p>
